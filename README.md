# StackCraft — Production-Grade Engineering Playbooks

## About

StackCraft is a platform for learning, building, and mastering production-grade software engineering — from foundations to architecture.

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Framer Motion

## Development

```sh
# Install dependencies
npm install

# Start development server
npm run dev
```

## Deployment

Deploy via the Publish button in the editor.
